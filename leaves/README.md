# üçÉ LEAVES: User-Facing Applications

The Leaves layer contains all user-facing applications that interact with the NWU Protocol.

## Components

### 1. Contributor Portal
- Submit data contributions
- Track submission status
- View rewards and reputation
- Manage profile and settings

### 2. Verifier Dashboard
- Review pending contributions
- Access verification tools
- Track verification history
- Earn verifier rewards

### 3. NFT Marketplace
- Trade contribution NFTs
- Browse verified data assets
- Auction and fixed-price sales
- Royalty distribution

### 4. Governance UI
- Create and vote on proposals
- View governance metrics
- Stake tokens for voting power
- Treasury management interface

### 5. Analytics Platform
- System-wide metrics
- Contribution trends
- Token economics dashboard
- Network health monitoring

## Technology Stack
- **Framework**: React 18 + Next.js 14
- **State Management**: Redux Toolkit
- **Web3 Integration**: ethers.js, wagmi
- **UI Components**: shadcn/ui, TailwindCSS
- **Charts**: Recharts, D3.js

## Development

```bash
cd leaves/[application-name]
npm install
npm run dev
```

See individual application READMEs for specific setup instructions.
